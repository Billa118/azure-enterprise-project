# What is Cloud Computing?

Cloud computing is the delivery of computing resources such as servers, storage, databases, and networking over the internet on a pay-as-you-go basis.

Instead of purchasing physical hardware, organizations rent infrastructure from cloud providers.

### Major Cloud Providers

- Microsoft Azure
- Amazon Web Services (AWS)
- Google Cloud Platform (GCP)

---

# Benefits of Cloud Computing

- Pay only for what you use
- High availability
- Easy scalability
- Global infrastructure
- Faster deployment
- Reduced maintenance

---

# What is Microsoft Azure?

Microsoft Azure is a cloud computing platform that allows organizations to build, deploy, and manage applications using Microsoft's global network of data centers.

Azure provides services such as:

- Virtual Machines
- Storage
- Networking
- Databases
- AI Services
- Monitoring
- Security

---

# Azure Regions

Azure resources are deployed into Regions.

Examples:

- Central India
- South India
- East US
- West Europe

### Why choose a nearby region?

- Lower latency
- Better performance
- Data compliance
- Disaster recovery planning

---

# Resource Group

A Resource Group is a logical container that holds related Azure resources.

Example:

EnterpriseProject-RG

Inside a Resource Group you can have:

- Virtual Machine
- Virtual Network
- Storage Account
- Network Security Group
- Key Vault

### Benefits

- Organize resources
- Apply RBAC permissions
- Track costs
- Manage resource lifecycle
- Delete related resources together

---

# Virtual Machine (VM)

A Virtual Machine is a software-based computer running in Azure.

A VM includes:

- CPU
- RAM
- Disk
- Operating System
- Network Interface

Common Operating Systems:

- Ubuntu
- Windows Server
- Red Hat Enterprise Linux

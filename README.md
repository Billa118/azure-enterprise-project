# Enterprise Azure Infrastructure Automation using Terraform & Azure DevOps

## Project Overview

This project demonstrates how to provision, secure, monitor, and deploy infrastructure on Microsoft Azure using Terraform and Azure DevOps.

The project simulates a real enterprise environment and covers Infrastructure as Code (IaC), CI/CD, monitoring, security, and application deployment.
